# V3 多质心版模型

## 模型信息

- **版本**: v3_multi_centroid
- **训练日期**: 2025-01
- **目标**: 多质心设计，提升多类型恶意检测

## 训练配置

- **Embedding模型**: BAAI/bge-small-en-v1.5 (384维)
- **投影维度**: 19维
- **训练轮数**: 300
- **优化器**: Adam (lr=0.01)

## 训练数据集

| 数据集 | 类型 | 样本数 |
|-------|------|-------|
| AdvBench | 恶意 | 520 |
| HarmBench | 恶意 | 400 |
| BeaverTails | 恶意 | 500 |
| 安全教育样本 | 正常 | 150 |
| 日常对话 | 正常 | 500 |

## 改进点

- 多恶意类别质心设计
- 同类聚集、异类分离损失函数
- 降低日常对话误报

## 测试结果

| 指标 | V2 | V3 |
|------|-----|-----|
| 恶意检出率 | ~90% | ~92% |
| 正常误报率 | ~12% | ~8% |

## 模型文件

- `learned_projection_weights_19d_multi_centroid.npy` - 投影矩阵
- `multi_centroid_config.json` - 多质心配置

## 问题

- 正常与恶意方向分离度不够
- 部分边界案例仍有误判
